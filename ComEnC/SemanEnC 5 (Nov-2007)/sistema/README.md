# SemanEnC 5 PHP Registration System

This code has been extracted to its own repository for standalone access:

**https://github.com/VitorMRodovalho/semanenc-php-registration**

PHP web application for online registration and certificate generation at the V Semana de Engenharia Civil, UEG Anapolis (November 2007). Features student registration, mini-course enrollment, and certificate generation in LaTeX/HTML/TXT formats.

The non-code conference files (photos, marketing, documents) remain in this repository under `ComEnC/SemanEnC 5 (Nov-2007)/`.
